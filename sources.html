In the 'What are Apes? - Orangutans' sub-section, here are the sources used to check each fact:


    Bornean Orangutans:

  • • Scientific Name: Pongo Pygmaeus
    • • https://www.worldwildlife.org/species/bornean-orangutan
    • • https://www.worldatlas.com/articles/orangutans-of-the-world-types-threats-and-conservation.html
    • • https://genomics.senescence.info/species/entry.php?species=Pongo_pygmaeus
    • • https://eol.org/pages/326450
  • • Height: 3.3-4.6 ft.
    • • https://www.worldwildlife.org/species/bornean-orangutan
    • • https://kidadl.com/animal-facts/bornean-orangutan-facts
    • • https://eol.org/pages/326450 - this source supports the upper end of this range since 1410mm≈4.6ft. As well as this, 1410mm is what they 'can grow to', however, later, height standing is put at 1160mm, which is within this range. In the citations, the site says that height standing for a female adult is 1160mm and for a male adult is 1400mm.
  • • Weight: 66-200 lbs
    • • https://www.worldwildlife.org/species/bornean-orangutan - this source says that they are between 66 and 220 pounds but I cannot find any other sources which support the extremes of these boundaries.
    • • https://www.bioexpedition.com/bornean-orangutan/ - this source supports the upper end of this range.
    • • https://web.archive.org/web/20090624114410/http://www.bbc.co.uk/nature/wildfacts/factfiles/360.shtml - this source supports the idea that most females are 37kg (≈82 pounds) and most males are 87kg (≈192 pounds). The idea that Bornean orangutans could be within 16 pounds (for females) and 8 pounds (for males) of these figures therefore does not sound too far fetched.
  • • Habitat: Lowland rainforests and tropical, swamp and mountain forests
    • • https://www.worldwildlife.org/species/bornean-orangutan
    • • https://animalia.bio/bornean-orangutan
    • • https://www.sciencedirect.com/topics/agricultural-and-biological-sciences/lowland-forests
  • • Population: In 2003, the highest estimate for the number of orangutans was 104, 700. This number has most likely declined by a significant amount since this time. One research study projected that by 2025, this would have declined to 47,000.
    • • https://www.worldwildlife.org/species/bornean-orangutan
    • • https://www.orangutanssp.org/uploads/2/4/9/9/24992309/2020-2022_safe_orangutan_program_plan.pdf
    • • https://www.iucnredlist.org/species/17975/123809220#population
  • • Fun Fact: Northeast Bornean orangutans are the smallest in size and found in Sabah and eastern Kalimantan as far as the Mahakam River.
    • • https://www.worldwildlife.org/species/bornean-orangutan

Image 1: This description of this image is the 'Aerial view of palm oil plantation on deforested land, Sabah, Borneo, Malaysia 2007'. This is taken from the source location which can be found via this link, and was provided by WWF Deutschland via Flickr. This image has been modified.

Every year, orangutans are killed/their homes are destroyed in order to make room for palm oil plantations. Through only buying products with sustainable palm oil, we can pressure companies into using sustainable palm oil (or none at all) in their products, helping to save the apes. For more information on this, please visit our What you can Do page, and learn about how you can do your part in helping to saving the apes.

Image 3: This description of this image is 'Aerial footage of palm oil and the forest in Sentabai Village, West Kalimantan. 2017'. This is taken from the source location which can be found via this link, and was provided by Nanang Sujana/CIFOR via Flickr.

Palm oil is a common ingredient in many household products. Used in all sorts of items, from chocolate to soap, for many companies, using palm oil is an easier way to make their products, however it comes at a cost. Each year, orangutans' homes are destroyed to make room for palm oil plantations such as these. If you would like to know more about what you can do to help stop this deforestation which is threatening the survival of orangutans, visit here, and learn about what you can do to help save the apes.
